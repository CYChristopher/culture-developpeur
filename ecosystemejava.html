Description

Le langage Java est un langage vaste qui permet de nombreuses possibilités. Toutefois, les 3 principaux sous-ensembles de ce Java sont : - Java SE qui permet de créer des applications client-lourd - Java EE qui permet de créer des applications web - Android qui permet de créer des applications mobiles La version actuelle de Java est la version 8.
